Branches

Em resumo, são ramos de um projeto ou documento de software (que é uma maneira mais segura de editar este arquivo) projetado para desenvolver em paralelo:

  • Correção de bugs;
  • Novas de funções;
  • Alteração do código principal.

    • O projeto pode ser colocado na Branch Master ou Main, que consiste no ramo primário de um repositório ou, então, em uma Branch Secundária, que seria uma ramificação desse repositório.

    Hospedagem

    O GitHub pages é utilizado para nossa hospedagem de site. Para sua utilização, devemos, inicialmente, escolher a aba pages dentro das settings do arquivo no GitHub, então, selecionamos qual Branch iramos enviar, com isso, é criado o site de hospedagem gratuita.
