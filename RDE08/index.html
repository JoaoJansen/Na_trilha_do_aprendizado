TAGS:

HTML Table:Cria uma tabela HTML que consiste em células de tabela dentro de colunas e linhas. Cada uma é definida pelos comandos: <td> e </td>. Tudo entre isso é o conteúdo da célula da tabela.

Del: A <del>tag define o trecho que foi excluído de um documento. Os navegadores riscam uma linha no texto excluído.
Enquanto o ins é a tag que faz uma marcação no texto inserido.

Sub: A <sub>tag define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha principal. Geralmente usado em fórmulas químicas.Sup: A <sup>tag define o texto sobrescrito. O texto sobrescrito aparece com meio caractere acima da linha principal. Geralmente usado em potências matemáticas.

Blockquote: A <blockquote>tag especifica uma seção citada de outra fonte.
Q: Já a <q>tag define uma pequena citação. Os navegadores normalmente inserem aspas entre aspas.

Abbr: A <abbr>tag define uma abreviatura ou acrônimo, como "HTML", "CSS", "Sr.", "Dr.", "ASAP", "ATM".

Address: A <address>tag define as informações de contato do proprietário de um documento. As informações geralmente são um endereço de e-mail, endereço físico, URL, identificador de mídia social e número de telefone. O texto no address é renderizado em itálico e os navegadores sempre adicionam uma quebra de linha antes e depois do comando address.

Cite: A <cite>tag define o título de um trabalho criativo. O texto nesse comando é processado em itálico .

BDO: BDO tem o significado de substituição bidirecional. A <bdo>tag é usada para substituir a direção do trecho atual.

Code: A <code>tag é utilizado para definir um pedaço de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Esta não possuii tanta relevância. Visto que é possível obter um resultado mais rico usando CSS.

Pre: A <pre>tag determina o texto pré-formatado. O texto que é usado em um comando pre é exibido em uma fonte de largura fixa e preserva os espaços e as quebras de linha. O texto será exibido exatamente como escrito no código-fonte HTML.

KBD: A <kbd>tag é usada para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.
